# Game Programming F21 Project - Geotris

## ✒️ Description

In this puzzle game, there will be shapes falling from the top of the world. The shapes are formed using multiple small square blocks and the shapes are of different forms. The player has to strategically place those falling shapes on the ground in the goal to make a connecting line of small square blocks.
